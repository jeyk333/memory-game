# Memory Game

### Game to check your memory power. Developed using Javascript.

## Usage
```
git clone https://github.com/jeyk333/memory-game.git
cd memory-game
//Start your local server
```
## JavaScript Methods and Functions Used
- querySelector()
- querySelectorAll()
- addEventListener()
- Math.random()
- appendChild()
- createElement()
- setAttribute()
- getAttribute()
- push()
- setTimeout()
- forloop
- alert()
- textContent()
- length
